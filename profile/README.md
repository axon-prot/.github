
**AXON** (Adaptive eXchange Over Negotiated‑context) is an open, binary, full‑duplex data transport protocol. It runs over any reliable transport (TCP, TLS, QUIC), negotiates encoding and compression per session, and provides built‑in primitives for capability discovery, semantic expiry, session fusion, state synchronization, and more.

The protocol is defined in the [axon‑prot/spec](https://github.com/axon-prot/spec) repository.
